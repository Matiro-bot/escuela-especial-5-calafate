# Instrucciones para la Aplicación Web Educativa

## Tecnología
- HTML, CSS, JavaScript vanilla

## Accesibilidad Obligatoria
- WCAG 2.1 nivel AA
- Navegación por teclado
- Atributos ARIA

## Estructura del Proyecto
- index.html
- carpeta css/
- carpeta js/

## Orientación Central del Proyecto
La aplicación integra actividades prácticas de TRES MATERIAS ESCOLARES: Biología, Matemática y Música. Cada materia tiene sus propios 3 niveles pedagógicos (Explorar, Resolver, Crear) con contenido específico.

## Estructura Rediseñada
- **Pantalla Principal**: Botones para seleccionar materia (Biología, Matemática, Música)
- **Selección de Materia**: Muestra los 3 niveles para esa materia (Explorar, Resolver, Crear)
- **Niveles por Materia**:
  - Explorar: Reconocimiento de patrones específicos de la materia
  - Resolver: Secuencias y problemas de la materia
  - Crear: Construcción propia en la materia

## Actividades por Materia

### Biología
- **Explorar**: Clasificar animales (vertebrados/invertebrados) en una cuadrícula
- **Resolver**: Ordenar las etapas de un ciclo de vida (mariposa, rana)
- **Crear**: Armar una cadena alimenticia con bloques (sol → planta → conejo → zorro)

### Matemática
- **Explorar**: Completar series numéricas (2, 4, 6, __, 10)
- **Resolver**: Ordenar pasos para resolver una ecuación simple (3 + __ = 7)
- **Crear**: Construir una figura geométrica con instrucciones (avanza 3, gira, avanza 2)

### Música
- **Explorar**: Reconocer patrones rítmicos (🔴🔴⚫🔴🔴⚫)
- **Resolver**: Ordenar secuencias de notas (do-re-mi → ?)
- **Crear**: Componer una melodía corta arrastrando notas
